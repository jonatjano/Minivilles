si vous êtes sous linux :
	lancer CreateVarEnv.sh // sauf si vous etes al'iut : lancer CreateVarEnvIUT.sh
	
	lancer Install.sh
	
	lancer startConsole.sh pour le mode consol et startGUI.sh pour le mode graphique
	
si vous êtes sous windows :
	
	crer une variable d'environnement système nommé 'MINIVILLE11' ayant pour valeur l'emplacement d'installation (dossier de l'appplication compris)
	
	lancer Install.bat
	
	lancer startConsole.bat pour le mode consol et startGUI.bat pour le mode graphique
